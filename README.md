# TIBEB V3

A rebuilt interactive editorial microsite for GitHub Pages.

## Highlights

- layered hero with small floating Ge'ez numerals
- floating 3D card with three scroll rotations
- organic ornament growth
- scroll-controlled illustration gallery
- pink Classic / Party page transition
- gamified Easter Egg hunt
- interactive card library with flip and modal views
- DM Mono via Google Fonts
- GSAP via jsDelivr
- responsive for mobile and desktop

## Publish

Upload the extracted contents to your existing GitHub repository and replace:
- `index.html`
- `css/`
- `js/`
- `assets/`

Then commit. GitHub Pages updates automatically.

## Edit content

Game rules and illustration texts:
- `js/main.js`

Main page text:
- `index.html`

Colors and layout:
- `css/styles.css`

## Sound

The `assets/audio` folder is prepared for a later sound update.
